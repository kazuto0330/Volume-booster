# Volume Booster (音量ブースター)

Webサイトの動画や音声の音量を、最大600%まで増幅できるChrome拡張機能です。
音が小さすぎる動画を視聴する際や、特定のサイトだけ音量を大きくしたい場合に便利です。

## 主な機能

*   **音量ブースト**: 10% ～ 600% の範囲で音量を調整可能。
*   **個別設定の保存 (Site-wide)**: ドメインやURLごとに音量設定を保存できます。一度設定すれば、次回アクセス時も自動で適用されます。
*   **一時的な調整 (Current Tab)**: 現在開いているタブの音量だけを一時的に変更できます。この設定は保存されません。
*   **ダークモード対応**: OSの設定や好みに合わせて、ライトモードとダークモードを切り替えられます。
*   **多言語対応**: 日本語と英語に対応しています。
*   **Web Audio API**: ChromeのWeb Audio APIを使用して、音質の劣化を抑えつつ音量を増幅します。

## インストール方法 (開発者モード)

この拡張機能はまだChromeウェブストアに公開されていないため、以下の手順でインストールしてください。

1.  このリポジトリをダウンロード（またはクローン）して、適当なフォルダに解凍します。
2.  Google Chromeを開き、アドレスバーに `chrome://extensions/` と入力してアクセスします。
3.  画面右上の **「デベロッパーモード」** スイッチをオンにします。
4.  左上に表示される **「パッケージ化されていない拡張機能を読み込む」** ボタンをクリックします。
5.  手順1で解凍したフォルダ（`manifest.json` が含まれているフォルダ）を選択します。
6.  拡張機能が一覧に追加されればインストール完了です。

## 使い方

### ポップアップ画面
ブラウザ右上の拡張機能アイコンをクリックすると、設定ポップアップが開きます。

*   **Site-wide (saved)**:
    *   現在閲覧中のサイト（ドメイン）に対する音量設定です。
    *   スライダーまたは数値入力で音量を変更すると、設定が自動的に保存されます。
*   **Current Tab (temporary)**:
    *   現在開いているタブのみに適用される一時的な設定です。
    *   タブを閉じたりリロードしたりするとリセットされます。
*   **アイコン**:
    *   ☀️/🌙: テーマ（ライト/ダーク）の切り替え。
    *   言語設定: 日本語/English の切り替え（設定メニュー内）。
    *   "すべての設定を管理" リンク: オプションページを開きます。

### オプションページ
登録済みのドメイン設定を一覧で管理できます。

*   **追加**: ドメイン名（例: `youtube.com`）とブースト率を入力して「追加」ボタンを押すと、手動で設定を追加できます。
*   **編集**: 一覧のブースト率を変更すると、即座に保存されます。
*   **削除**: ゴミ箱アイコンをクリックすると、そのドメインの設定を削除します。
*   **リセット**: 「すべての設定をリセット」ボタンで、保存されているすべての設定を消去します。

## 技術的な詳細

*   **Manifest V3**: 最新のChrome拡張機能仕様（Manifest V3）に準拠しています。
*   **Web Audio API**: `AudioContext`, `MediaElementSource`, `GainNode` を使用してオーディオ信号を処理しています。
*   **MutationObserver**: 動的に追加された動画や音声要素（YouTubeの自動再生など）も検知してブーストを適用します。
*   **Storage API**: `chrome.storage.sync` を使用して、デバイス間で設定を同期します。

## 権限について

*   `storage`: 設定（音量、テーマ、言語）を保存するために使用します。
*   `scripting`: Webページに音量制御スクリプトを注入するために使用します。
*   `activeTab` / `<all_urls>`: どのWebサイトでも音量を制御できるようにするために必要です。ユーザーの閲覧履歴や個人情報を収集することはありません。
